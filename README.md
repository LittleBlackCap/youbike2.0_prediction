# youbike2.0_prediction
Predict future bike availability at YouBike stations using historical data.  
藉由台北、台中、高雄市政府提供之公開資料，預測特定站點哪個時段最容易缺車或滿車。

### Problem  
在租借腳踏車時發現有的時段某站就是沒有車無法借車、或是某站還滿車，於是想找出到底哪些時候要避開去某些站借車或還車，並在地圖上顯示。
